# Система бронирования ресторана

Учебный проект на Python с использованием PostgreSQL.  
Реализует систему бронирования столов в ресторане с графическим интерфейсом (tkinter).

Проект выполнен в рамках задания на отработку:
- CRUD-операций
- работы с PostgreSQL
- проверки пересекающихся бронирований
- простого GUI

---

## Функциональность

### Пользователи
- Создание пользователя
- Просмотр списка пользователей
- Обновление данных пользователя
- Удаление пользователя

### Столы
- Создание стола
- Просмотр списка столов
- Обновление данных стола
- Удаление стола

### Бронирования
- Создание бронирования
- Просмотр списка бронирований
- Обновление бронирования
- Удаление бронирования
- **Проверка доступности стола**  
  (запрет пересекающихся бронирований по одному столу)

---

## Проверка доступности

Перед созданием бронирования выполняется проверка:
Стол считается занятым, если существует бронирование, которое пересекается по времени:

existing.start < new_end AND existing.end > new_start

В GUI доступна отдельная кнопка **«Проверить доступность»**.

---

## Графический интерфейс (GUI)

Приложение использует `tkinter` и содержит вкладки:
- Пользователи
- Столы
- Бронирования

Все операции (CRUD и проверка доступности) выполняются через GUI.

## Установка

1. Клонировать репозиторий или распаковать архив
2. Установить зависимости:

```bash
pip install -r requirements.txt
Настройка базы данных
Создайте файл .env в корне проекта:

DB_HOST=localhost
DB_PORT=5432
DB_NAME=postgres
DB_USER=postgres
DB_PASS=your_password
Файл .env.example содержит пример без секретов.

Запуск приложения
python main.py или
python gui.py

При запуске:

- таблицы создаются автоматически, если их нет

- применяются SQL-миграции

- открывается графический интерфейс

Используемые технологии:
Python 3
PostgreSQL
psycopg2
